Надо добавить фильтр по категории в раздел "Архив". Бэкенд его поддерживает.
Поле должно выбираться из списка категорий пользователя.
Добавь поле-фильтр и добавляй его в json при отправке на бэкенд - ключ "category".
Поля которые необходимо передать - Long id; String name; Type type;



На странице "Операции" для каждой операции без назначенной категории (отображаются по центру) с бэкенда приходит suggestedCategoryId.
см src/main/java/com/override/dto/TransactionDTO.java. Это айди подсказки, которую мы расчитали на бэкенде.
При перетаскивании транзакции - надо подсветить фиолетовым категорию у которой id равен suggestedCategoryId.
Не для всех транзакций есть подсказки, в таком случае подсвечивать ничего не нужно.

На странице "Операции" блоки с операциями по центру содержат иконку "карандаш".
При клике на эту иконку надо открывать модальное окно, которое используется на странице "архив" для редактирования транзакций.
Вынеси это модальное окно в отдельный компонент.
