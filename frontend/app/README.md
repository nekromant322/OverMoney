# OverMoney

## Требования
- [NodeJS 20+](https://nodejs.org)

## Рекомендации
- [VSCode](https://code.visualstudio.com/)
- [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## Запуск
1. `npm install`
2. `npm run dev`

## Сборка
1. `npm install`
2. `npm run lint`
3. `npm run build`
